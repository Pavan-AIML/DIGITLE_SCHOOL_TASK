<h1>Building a regression prediction model for the number of deaths in a given month and year </h1>
<h3> To build this application twe will use Linear Regression as it is a regression problem with the output node number of accidents and input node will be year and month. </h3>
<h4> Tools used in this project. </h4>

- Python for coding.
- Fast API to build the app instance.
- EC2 AWS instance to deploy the model.
- Also used streamlit for the front-end and the Streamlit server for deploying the model.
- Used data set https://opendata.muenchen.de/dataset/monatszahlen-verkehrsunfaelle/resource/40094bd6-f82d-4979-949b-26c8dc00b9a7
